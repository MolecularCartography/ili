Common operations

Action Shortcut Alternative
Open files Ctrl+O Drag and drop files to the browser window
Switch between molecular maps Ctrl+↑/↓ Click on a name of an active map above the colorbar
Find a molecular map by its name Ctrl+F Click on a name of an active map above the colorbar and start typing.
Make a snapshot Ctrl+S NA

Note: OS X users should use ⌘ key instead of Ctrl.


Image manipulations

Action How-to Views where available

Move image Move mouse holding left button 2D
Zoom Spin mouse wheel 2D, 3D
Display spot name Left-click at a spot 2D, 3D
Move model Move mouse holding right button 3D
Enable/disable model auto-rotation Double-click 3D
